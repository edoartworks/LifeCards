# LifeCards
 is about connecting deeper with friends, lovers 
or strangers.

Ask questions.
Answer or skip questions.
Pass the phone.

Respect others' boundaries,
and your own boundaries.

Have courage
to open yourself up, 
be vulnerable.
Dare to trust.
